# openclaw 安装技能



### **步骤1：解压zip文件**                                                                                                                                                                              

 ```bash                                                                                                                
   # 进入OpenClaw工作目录                                                                                               
   cd /home/zap/.openclaw
                                                                                                                        
   # 解压zip文件                                                                                                        
   unzip skill-vetter-1.0.0.zip -d skill-vetter-temp
 ```

 ### 步骤2：检查解压后的内容                                                                                            

 ```bash                                                                                                                
   # 查看解压后的文件结构                                                                                               
   ls -la skill-vetter-temp/                                                                                            
                                                                                                                        
   # 检查是否有SKILL.md文件                                                                                             
   ls -la skill-vetter-temp/SKILL.md                                                                                    
 ```

 ### 步骤3：复制到技能目录 （建议放到用户目录）     

 ```bash                                                                                                                
   # 复制到OpenClaw技能目录（需要sudo权限）                                                                             
   sudo cp -r skill-vetter-temp/* /usr/lib/node_modules/openclaw/skills/skill-vetter/
                                                                                                                        
   # 或者复制到用户目录（如果不想用sudo）                                                                               
   mkdir -p ~/.openclaw/skills/skill-vetter
   cp -r skill-vetter-temp/* ~/.openclaw/skills/skill-vetter/
 ```

### 步骤4：清理临时文件                                                                                                                                                                                                                        

 ```bash                                                                                                                
   rm -rf skill-vetter-temp
 ```

如果你选择安装到 ~/.openclaw/skills/，可能需要配置OpenClaw来查找用户技能目录。检查OpenClaw配置文件：                   
 ```bash                                                                                                                
   cat ~/.openclaw/openclaw.json
 ```

 在配置中可能需要添加用户技能路径 ；编辑 ~/.openclaw/openclaw.json，添加：                                                                                                     

 ```json                                                                                                                
   {                                                                                                                    
     "skills": {                                                                                                        
       "paths": [                                                                                                       
         "~/.openclaw/skills"                                                                                           
       ]                                                                                                                
     }                                                                                                                  
   }     
 ```

重启OpenClaw验证！！

### 如何验证？

执行命令：显示ready说明可用

```
openclaw skills list
或者 
penclaw skills info <技能名>
```
